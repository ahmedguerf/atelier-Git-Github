Démo pratique - Partie 2

LIVEConnexion à GitHub

1

Créer Repo

Sur GitHub, créer un nouveau repository. Ne pas initialiser avec README.

2

Lier Distant

Connecter le local au serveur :

git remote add origin URL
3

Renommer

Standardiser le nom de la branche principale :

git branch -M main
4

Push

Envoyer le code et définir le suivi :

git push -u origin main
5

Vérifier

Rafraîchir la page GitHub. Le code et l'historique sont maintenant en ligne !

Objectif : Local ↔ Distant fonctionne !
