Commandes de branches

Travailler en parallèle

bash — git-workflow
# 1. Voir les branches
$git branch
# 2. Créer une nouvelle branche
$git branch ma-fonctionnalite
# 3. Changer de branche
$git switch ma-fonctionnalite
# 4. Créer ET changer (Raccourci)
$git switch -c nouvelle-branche
# 5. Fusionner une branche
$git merge ma-fonctionnalite

Astuce : Créez toujours une branche pour chaque nouvelle fonctionnalité !
