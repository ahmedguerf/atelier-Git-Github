Vocabulaire de base

Les 6 mots à retenir

REPOSITORY

Le dossier de ton projet qui contient tous tes fichiers et l'historique des modifications (sur Git ou GitHub).

STOCKAGE

COMMIT

Une photo instantanée de ton code à un moment précis. C'est un point de sauvegarde avec un message.

ACTION

BRANCHE

Une timeline parallèle pour tester des idées ou développer une fonctionnalité sans casser le projet principal.

STRUCTURE

MERGE

L'action de fusionner deux branches ensemble (par exemple, intégrer ta fonctionnalité finie dans le projet principal).

ACTION

PUSH

Envoyer son code et son historique depuis ton ordinateur (Git local) vers le serveur (GitHub distant).

SYNC ⬆️

PULL

Récupérer le code et les mises à jour depuis le serveur (GitHub distant) vers ton ordinateur.

SYNC ⬇️
