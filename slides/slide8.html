Les 5 commandes essentielles

bash — workflow
# 1. Démarrer un projet $git init
# 2. Voir ce qui se passe (état des fichiers) $git status
# 3. Préparer des fichiers (Staging) $git add .
# 4. Sauvegarder (avec message) $git commit -m "Message clair"
# 5. Voir l'historique $git log --oneline
$

Astuce : Utilisez ces commandes dans cet ordre pour démarrer !
