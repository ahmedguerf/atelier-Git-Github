GitHub : Qu'est-ce que c'est ?

Définition complète

GitHub = Plateforme en ligne qui héberge des dépôts Git

Service de stockage cloud

Comme Google Drive, mais spécialisé pour le code et l'historique.

Réseau social dev

Follow des développeurs, like (star) des projets, commenter du code.

Plateforme de collaboration

Outils pour travailler en équipe sur le même projet sans chaos.

Portfolio professionnel

Ta vitrine pour montrer tes projets et ton code aux recruteurs.

Git = l'outil

GitHub = le réseau social
